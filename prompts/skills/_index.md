# Индекс Skills

Этот файл содержит описания всех доступных skills для автоматического выбора.

## Как выбрать skill

Проанализируй запрос пользователя и определи, какой skill лучше подходит.
Если ни один специализированный skill не подходит — используй базовый промпт.

## Доступные skills

### dossier
**Файл:** `skills/dossier.md`
**Триггеры:** досье, профиль пользователя, кто такой, информация о пользователе
**Когда использовать:** Когда пользователь спрашивает о конкретном человеке

### summary
**Файл:** `skills/summary.md`
**Триггеры:** саммари, дайджест, о чём говорили, что обсуждали, главное за период
**Когда использовать:** Когда нужен обзор обсуждений за период времени

### search
**Файл:** `skills/search.md`
**Триггеры:** найди, поиск, где упоминается, кто писал про
**Когда использовать:** Когда нужно найти конкретные сообщения по ключевым словам

### top
**Файл:** `skills/top.md`
**Триггеры:** топ, рейтинг, самые активные, популярные, больше всех
**Когда использовать:** Когда нужны рейтинги и статистика по метрикам

## Формат использования в боте

```python
SKILLS = {
    "dossier": {
        "triggers": ["досье", "профиль", "кто такой", "кто такая", "информация о"],
        "file": "skills/dossier.md"
    },
    "summary": {
        "triggers": ["саммари", "дайджест", "о чём говорили", "что обсуждали", "главное за"],
        "file": "skills/summary.md"
    },
    "search": {
        "triggers": ["найди", "поиск", "где упоминается", "кто писал про"],
        "file": "skills/search.md"
    },
    "top": {
        "triggers": ["топ", "рейтинг", "самые активные", "популярные", "больше всех"],
        "file": "skills/top.md"
    }
}
```
